# Portfolio
A modern, responsive landing page built with pure HTML, CSS, and JavaScript. Designed to showcase web development services with a strong focus on conversion, performance, and clean user experience.

## Features
- Responsive layout
- Modern UI/UX
- Hero section with call-to-action
- Services and benefits sections
- Business niche showcase
- Before vs After comparison
- Conversion-focused CTA
- Scroll reveal animations
- Smooth scrolling navigation
- Clean and organized structure
- No frameworks
- No build tools

## Technologies
- HTML5
- CSS3
- JavaScript (Vanilla)

## Getting Started
Clone the repository:
```bash
git clone [repo-link]
```

Open the project folder:
```bash
cd [repo-name]
```

Run the project by opening `index.html` in your browser (No installation or dependencies are required.)
